# 📊 Sales Data Analysis (Python)

This project analyzes retail sales data using **Python, Pandas, and Matplotlib** to uncover key business insights.

---

## 📁 Dataset
- Superstore sales dataset
- Contains order details, products, regions, and revenue

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib

---

## 📊 Analysis Performed
- Total sales calculation
- Top 5 best-selling products
- Sales by region
- Sales by category

---

## 📈 Visualization
![Sales Chart](sales_chart.png)

---

## 🔍 Key Insights
- Technology generates the highest revenue
- Canon Copier is the top-performing product
- West region has the highest sales
- Sales are concentrated in a few high-value items
