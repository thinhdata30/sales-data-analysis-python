# sales-data-analysis-python
Sales data analysis using Python (Pandas, Matplotlib). Includes insights on top products, regions, and categories.

# Sales Data Analysis (Python)

This project analyzes sales data using **Python, Pandas, and Matplotlib** to uncover key business insights.


## Dataset
- Superstore sales dataset
- Contains information on orders, products, regions, and revenue


## Tools & Technologies
- Python
- Pandas (data analysis)
- Matplotlib (data visualization)


## Key Analysis
- Total sales calculation
- Top 5 best-selling products
- Sales by region
- Sales by category


##  Visualization
![Sales Chart](Sales_chart.png)


##  Key Insights
- Technology generates the highest revenue
- Canon Copier is the top-performing product
- Sales are concentrated in a few high-value items
- West region has the highest sales

pip install pandas matplotlib
